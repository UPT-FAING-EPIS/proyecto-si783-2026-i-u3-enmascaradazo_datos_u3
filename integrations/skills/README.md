# Skills de Enmask

Este directorio contiene dos Skills empaquetados para la entrega academica.

## 1. Enmask Data Protection Advisor

Ubicacion:

```text
enmask-data-protection-advisor/skill.zip
```

Uso:

- Recomendar reglas de enmascaramiento.
- Seleccionar algoritmos por tipo de dato.
- Elegir modo de proteccion.
- Generar documentacion tecnica y evidencias.

## 2. Enmask Connection Diagnostician

Ubicacion:

```text
enmask-connection-diagnostician/skill.zip
```

Uso:

- Diagnosticar errores de conexion.
- Guiar configuraciones de SQL Server, Oracle, Redis, Cassandra, Neo4j, MongoDB, PostgreSQL y MySQL.
- Explicar problemas de URI cloud, TLS, puertos, drivers y autenticacion.

## Nota

Estos archivos se adjuntan al repositorio junto al proyecto. Si se requiere publicacion externa, se suben a la biblioteca de Skills correspondiente y se documenta la evidencia en GitHub.
